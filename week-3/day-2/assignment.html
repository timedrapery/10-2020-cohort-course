Mouse Events

We're going to cover the most frequently used mouse events for final projects

Event Name Code Example Fired When
click <button onclick="buttonClick()">Click me</button> Fires on a mouse click on the element
dblclick <button ondblclick="doublebuttonClick()">Double click me </button> Fires on a mouse double-click on the element
mouseover <button mouseover="mouseOverEvent()">hover me</button> Fires when the mouse pointer moves over an element
mouseout <button mouseover="mouseOverEvent()">hover and leave</button> Fires when the mouse pointer moves out of an element
drag w3schools Example Script to be run when an element is dragged
